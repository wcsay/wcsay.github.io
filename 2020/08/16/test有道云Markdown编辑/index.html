test有道云Markdown编辑

2020-08-16

有道云笔记是支持markdown编辑的，并且支持实时预览，现在就测试一下看看效果如何，现在我用的编辑软件是markdown2，这个编辑器太简陋了，运行的速度也很慢，我不是很满意，其实notepad2也支持markdown编辑，这个速度和打开记事本一样，非常快，可是没有实时预览，鱼与熊掌不可兼得吧。
试用了一下，这个编辑器需要另存为，没有直接保存的功能，ciao！
